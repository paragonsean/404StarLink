Get a branded food item by name (foodBrandedNamePhpGet)

Search for branded food items by name.

Example

https://chompthis.com/api/v2/food/branded/name.php?api_key=API_KEY&name=NAME

Tips

  • Get started by using our food lookup tool.

This API endpoint is only available to Standard and Premium API subscribers. Please consider upgrading your subscription if you are subscribed to the Limited plan. Read this if you aren't sure how to upgrade your subscription.

Query parameters

name (required)
Query Parameter —

Search for branded food items using a general food name keyword. This does not have to exactly match the "official" name for the food.

Example

&name=Starburst

default: null
limit (optional)
Query Parameter —

Set maximum number of records you want the API to return. The default value is "10."

Example

&limit=10

default: null
page (optional)
Query Parameter —

This is how you paginate the search result. By default, you will see the first 10 records. You must increment the page number to access the next 10 records, and so on. The default value is "1."

Example

&page=1

default: null

allergen (optional)
Query Parameter —

&allergen=Peanuts
