Xero Bank Feeds API

The Bank Feeds API is a closed API that is only available to financial institutions that have an established financial services partnership with Xero. If you're an existing financial services partner that wants access, contact your local Partner Manager. If you're a financial institution who wants to provide bank feeds to your business customers, contact us to become a financial services partner.
More information: https://developer.xero.com
Contact Info: api@xero.com
Version: 2.9.4
BasePath:/bankfeeds.xro/1.0

FeedConnection - Up

accountId (optional)
UUID Xero identifier for a bank account in Xero. Must be included if AccountNumber is not specified. format: uuid
accountName (optional)
String The Account Name will be used for the creation of a new Xero Bank Account if a matching Xero Bank Account is not found.
accountNumber (optional)
String String(40) when AccountType is BANK String(4) when AccountType is CREDITCARD The Account Number is used to match the feed to a Xero Bank Account. The API will create a new Xero Bank Account if a match to an existing Xero Bank Account is not found. Only the last 4 digits must be supplied for Credit Card numbers. Must be included if AccountId is not specified.
accountToken (optional)
String This account identifier is generated by the financial institute (FI). This must be unique for your financial institute.
accountType (optional)
String High level bank account type - BANK CREDITCARD BANK encompasses all bank account types other than credit cards.
