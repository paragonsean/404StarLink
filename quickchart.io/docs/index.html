get /chart
Generate a chart (GET) (chartGet)
Generate a chart based on the provided parameters.

Query parameters

Query Parameter — The chart configuration in Chart.js format (JSON or Javascript). default: null
width (optional)

height (optional)

format (optional)
Query Parameter — The output format of the chart, e.g., 'png', 'jpg', 'svg', or 'webp'. default: null
