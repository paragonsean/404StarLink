Analytics

Up 
get /analytics
(analytics)

Query parameters

start (optional)
Query Parameter — Start date of the statistics in the format YYYY-MM-DD. By default, the date of 30 days ago is set. default: null
end (optional)
Query Parameter — End date of the statistics in the format YYYY-MM-DD. By default, the current day. default: null
label (optional)
Query Parameter — Shows only data of a specific label. default: null
subaccounts (optional)
Query Parameter — Receive the data only for the main account, all your (sub-)accounts or only for specific subaccounts. default: null
group_by (optional)
Query Parameter — Defines the grouping of the data. default: null

Return type

Analytics_200_response

Example data

Content-Type: application/json
{
  "date" : "date",
  "voice" : 7,
  "inbound" : 5,
  "direct" : 0,
  "economy" : 6,
  "usage_eur" : 2.302136,
  "mnp" : 5,
  "hlr" : 1
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Query parameters

Query Parameter — The actual text message to send. default: null

Query Parameter — The recipient number or group name. default: null

Query Parameter — Set a custom sender name. default: null

Query Parameter — Identifier to return in callbacks. default: null
label (optional)
Query Parameter — A custom label. default: null
udh (optional)
Query Parameter — A custom User Data Header. default: null
delay (optional)
Query Parameter — Date/Time for delayed dispatch. default: null
debug (optional)
Query Parameter — Disable message sending. default: 0
no_reload (optional)
Query Parameter — Enable sending of duplicated messages within 180 seconds. default: 0
unicode (optional)
Query Parameter — Force unicode encoding. Reduces sms length to 70 chars. default: 0
flash (optional)
Query Parameter — Send as flash. default: 0
utf8 (optional)
Query Parameter — Force UTF8 encoding. default: 0
details (optional)
Query Parameter — Attach message details to response. default: 0
return_msg_id (optional)
Query Parameter — Attach message ID to second row in a text response. default: 0
json (optional)
Query Parameter — Return a detailed JSON response. default: 0
performance_tracking (optional)
Query Parameter — Enable performance tracking for found URLs. default: 0
