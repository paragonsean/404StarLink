# HlsEncryptionType

Encrypts the segments with the given encryption scheme. Leave blank to disable. Selecting 'Disabled' in the web interface also disables encryption.

## Enum

* `AES128` (value: `'AES128'`)

* `SAMPLE_AES` (value: `'SAMPLE_AES'`)
