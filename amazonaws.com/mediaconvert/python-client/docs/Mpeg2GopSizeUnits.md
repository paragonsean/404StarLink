# Mpeg2GopSizeUnits

Specify the units for GOP size (GopSize). If you don't specify a value here, by default the encoder measures GOP size in frames.

## Enum

* `FRAMES` (value: `'FRAMES'`)

* `SECONDS` (value: `'SECONDS'`)
