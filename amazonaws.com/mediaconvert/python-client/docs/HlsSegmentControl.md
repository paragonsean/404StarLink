# HlsSegmentControl

When set to SINGLE_FILE, emits program as a single media resource (.ts) file, uses #EXT-X-BYTERANGE tags to index segment for playback.

* `SINGLE_FILE` (value: `'SINGLE_FILE'`)

* `SEGMENTED_FILES` (value: `'SEGMENTED_FILES'`)
