# Av1AdaptiveQuantization

Specify the strength of any adaptive quantization filters that you enable. The value that you choose here applies to Spatial adaptive quantization (spatialAdaptiveQuantization).

## Enum

* `FALSE` (value: `'false'`)

* `LOW` (value: `'LOW'`)

* `MEDIUM` (value: `'MEDIUM'`)

* `HIGH` (value: `'HIGH'`)

* `HIGHER` (value: `'HIGHER'`)

* `MAX` (value: `'MAX'`)
