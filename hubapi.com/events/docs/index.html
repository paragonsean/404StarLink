get /events/v3/events
Returns a collection of events matching a query. (get/events/v3/eventsGetPage)

Query parameters

occurredAfter (optional)
Query Parameter — The starting time as an ISO 8601 timestamp. default: null format: date-time
occurredBefore (optional)
Query Parameter — The ending time as an ISO 8601 timestamp. default: null format: date-time
objectType (optional)
Query Parameter — The type of object being selected. Valid values are hubspot named object types (e.g. contact). default: null
objectId (optional)
Query Parameter — The id of the selected object. If not present, then the objectProperty parameter is required. default: null format: int64
eventType (optional)
Query Parameter — Limits the response to the specified event type. For example &eventType=e_visited_page returns only e_visited_page events. If not present all event types are returned. default: null
after (optional)
Query Parameter — An additional parameter that may be used to get the next limit set of results. default: null
before (optional)
Query Parameter — default: null
limit (optional)
Query Parameter — The maximum number of events to return, defaults to 20. default: null format: int32
sort (optional)
Query Parameter — Selects the sort field and order. Defaults to ascending, prefix with - for descending order. occurredAt is the only field supported for sorting. default: null

Return type

CollectionResponseExternalUnifiedEvent
