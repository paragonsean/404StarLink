get /orders
List orders (ordersGet)
Gets a paginated list of all orders.

Query parameters

orderStatus (optional)
Query Parameter — Filter by status default: null
offset (optional)
Query Parameter — The number of items to skip before starting to collect the result set. default: null
limit (optional)
Query Parameter — The maximum number of items to return. If the value exceeds the maximum, then the maximum value will be used. default: null

Return type

ListOfOrders
