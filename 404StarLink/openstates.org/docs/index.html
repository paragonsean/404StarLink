Bills Search (billsSearchBillsGet)

Search for bills matching given criteria.

Must either specify a jurisdiction or a full text query (q). Additional parameters will futher restrict bills returned.

Request headers

x-api-key (optional)
Header Parameter — default: null

Query parameters

jurisdiction (optional)
Query Parameter — Filter by jurisdiction name or ID. default: null
session (optional)
Query Parameter — Filter by session identifier. default: null
chamber (optional)
Query Parameter — Filter by chamber of origination. default: null
identifier (optional)
Query Parameter — Filter to only include bills with this identifier. default: []
classification (optional)
Query Parameter — Filter by classification, e.g. bill or resolution default: null
subject (optional)
Query Parameter — Filter by one or more subjects. default: []
updated_since (optional)
Query Parameter — Filter to only include bills with updates since a given date. default: null
created_since (optional)
Query Parameter — Filter to only include bills created since a given date. default: null
action_since (optional)
Query Parameter — Filter to only include bills with an action since a given date. default: null
sort (optional)
Query Parameter — Desired sort order for bill results. default: updated_desc
sponsor (optional)
Query Parameter — Filter to only include bills sponsored by a given name or person ID. default: null
sponsor_classification (optional)
Query Parameter — Filter matched sponsors to only include particular types of sponsorships. default: null
q (optional)
Query Parameter — Filter by full text search term. default: null
include (optional)
Query Parameter — Additional information to include in response. default: []
page (optional)
Query Parameter — default: 1
per_page (optional)
Query Parameter — default: 10
apikey (optional)
