get /self/calls
List active calls (listCalls)
Lists currently active calls

Query parameters

fromDate (optional)
Query Parameter — Return calls that occurred after this point in time default: null
toDate (optional)
Query Parameter — Return calls that occurred before this point in time default: null
direction (optional)
Query Parameter — Filter by call direction. For multiple criteria, seperate values by a comma. default: null
states (optional)
Query Parameter — Filter calls by state. For multiple criteria, seperate values by a comma. default: ACTIVE
offset (optional)
Query Parameter — Page number of calls to return default: null format: int64
size (optional)
Query Parameter — Return this amount of calls in the response default: 20
order (optional)
Query Parameter — Sort in either ascending or descending order default: ASC
sort (optional)
Query Parameter — Sort calls by property default: null
