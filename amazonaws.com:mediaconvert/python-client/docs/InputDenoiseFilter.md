# InputDenoiseFilter

Enable Denoise (InputDenoiseFilter) to filter noise from the input. Default is disabled. Only applicable to MPEG2, H.264, H.265, and uncompressed video inputs.

## Enum

* `ENABLED` (value: `'ENABLED'`)

* `DISABLED` (value: `'DISABLED'`)
