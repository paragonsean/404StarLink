# HlsOfflineEncrypted

Enable this setting to insert the EXT-X-SESSION-KEY element into the master playlist. This allows for offline Apple HLS FairPlay content protection.

## Enum

* `ENABLED` (value: `'ENABLED'`)

* `DISABLED` (value: `'DISABLED'`)
