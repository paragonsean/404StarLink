# Mpeg2RateControlMode

Use Rate control mode (Mpeg2RateControlMode) to specify whether the bitrate is variable (vbr) or constant (cbr).

## Enum

* `VBR` (value: `'VBR'`)

* `CBR` (value: `'CBR'`)
