# MsSmoothManifestEncoding

Use Manifest encoding (MsSmoothManifestEncoding) to specify the encoding format for the server and client manifest. Valid options are utf8 and utf16.

## Enum

* `UTF8` (value: `'UTF8'`)

* `UTF16` (value: `'UTF16'`)
